# StartIE

 ## what is this?

launches the classic Internet Explorer using the classic Win32 API on modern Windows.  
may work on Windows XP or later (unconfirmed)

## requirements
* Windows SDK
* ATL

## usage
just run the command or double click the icon.  
no parameters required.
```
> StartIE
```
